# UEFI Support for L4Re

This package contains a port of gnu-efi together with make files for building
it inside the L4Re tree, used for generating bootable UEFI images.

# Documentation

This package is part of the L4Re operating system. For documentation and
build instructions please refer to [l4re.org](https://l4re.org).
